# 🚀 AI Email Reply Generator

An AI-powered email reply generator built using **Spring Boot + Gemini API + Chrome Extension + React**.  
This project helps users generate smart email replies instantly inside Gmail.

---

## 💡 Features

- 📩 Reads email content directly from Gmail
- 🤖 Generates AI-based replies using Gemini API
- ⚡ One-click "AI Reply" button inside Gmail
- ✍️ Automatically inserts reply into compose box
- 🌐 Web interface also available

---

## ⚙️ How It Works

1. User opens Gmail
2. Chrome Extension injects "AI Reply" button
3. User clicks button
4. Email content is captured
5. Request sent to Spring Boot backend
6. Backend calls Gemini API
7. AI generates response
8. Reply inserted into Gmail compose box
